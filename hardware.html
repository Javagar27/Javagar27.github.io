Hardware in een computer

Een computer is een een van de beste uitvindingen van de mens. Het heeft heel veel functies: Je kan er mee rekenen,video's afspelen en bewerken,gamen en nog veel meer. Dit doet een computer niet zomaar. Het heeft veel verschillende onderdelen nodig om dit te doen. Je kan deze onderdelen in twee hoofdcategoriën indelen: Software en hardware. Software zijn allerlei programma's zoals Windows 10. Hardware zijn onderdelen die je echt kunt voelen. Dus denk aan je Harddrives,CPU's en GPU's.
Hier ga ik het over 2 hardware onderdelen hebben.

Harde schijf

de harde schijf,ook wel HDD(Engels: Hard Disk Drive) is een vorm van een extern geheugen onderdeel,waar een computer allemaal data kan opslaan.
Op 13 september 1956 bracht IBM de eerste harde schijf op de markt: Random Access Method of Accounting and Control. De RAMAC bestond uit 50 gestapelde magnetische schijven . De totale opslag van deze schijf was 5 MB.
Er zijn veel voordelen van een harde schijf: Je kan er tegenwoordig veel data op opslaan. En het is permanente data dat je opslaat in tegenstelling tot RAM opslag. Soms kan je harde schijven ook meenemen en op andere computers aansluiten. Je kan dan data up- of downloaden hierop. Het nadeel is wel dat als je een harde schijf wilt verkopen,je misschien per ongeluk niet alles hebt kunnen wissen. Er kunnen misschien vertrouwelijke gegevens opzitten waar een ander persoon toegang tot kan krijgen.
Je kan vele soorten harde schijven tegenwoordig op de markt kopen in allerlei soorten en maten. Je kan kiezen uit compacte schijven of schijven die héél veel data op kan slaan.
De harde schijf heeft vele ontwikkelingen doorgelopen. Vooral de opslag capaciteit. De eerste had een opslagcapaciteit van 5MB. Nu kan je schijven van wel 5TB halen of meer.

moederbord

Om een PC in elkaar te zetten heb je natuurlijk iets nodig waarop je alle elektronica kan aansluiten en laten werken. Hier hebben wij het moederbord voor.
Op een moederbord kan men allerlei componenten vinden zoals: CPU's,GPU's,connectors voor uitbreidingskaarten,RAM's etc.
